Steps to run the project:

1.Open the solution in Visual Studio 2019.
2.In Solution Explorer, right click on ShopBridge Project and go to Properties.
3.In Debug -> check the launch browser and add swagger in the text box.
4.Run the solution. Swagger page will open.
5.On the swagger page on browser -> go to ShopBridge
6.There will be 4 Http requests (Get, Post, Put, Delete)
7.To fetch all the product details click on Get, for Adding product record click on Post, for updation click on Put and for deleting record click on Delete.
8.The database script folder is also uploaded on github repository.
